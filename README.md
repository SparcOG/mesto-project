# Mesto Russia

Одностраничный сайт посвященный путешествиям по России. В основной функционал входит адаптивная верстка с построением сетки Grid Layout, липкий футер и модальное окно.

#### Использовались следующие технологии:

* BEM

* Flexbox

* Grid Layout

* Positioning

* Pseudoclasses

* Pseudo elements

* Media Queries
